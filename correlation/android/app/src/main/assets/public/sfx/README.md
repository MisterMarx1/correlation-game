# Sound Effects for NULL Game

## Free Sound Effect Resources

1. **Mixkit** - https://mixkit.co/free-sound-effects/game/
   - Royalty-free game sounds
   - No attribution required
   - Categories: hits, coins, game over, power-ups

2. **Pixabay** - https://pixabay.com/sound-effects/search/game/
   - Free game sound effects
   - Commercial use allowed
   - Download in MP3 format

3. **Freesound** - https://freesound.org/
   - Large community database
   - Check licenses for each sound
   - High-quality variety

## Recommended Sounds for NULL Game

### Core Actions
- **Select/Click**: Short click sound when selecting cells
- **Correlation Success**: Energy transfer sound (electric/quantum)
- **Decorrelation/Failure**: Error/buzz sound
- **Recoil Cascade**: Explosion/impact sound

### Game Events
- **Game Start**: Ambient startup sound
- **Game Over**: Dramatic end sound
- **Level Up**: Positive progression sound
- **Combo Multiplier**: Special effect for cascades

### UI Elements
- **Menu Navigation**: Soft click sounds
- **Pause**: Subtle transition sound
- **Spawn**: Energy cell appearance sound

## Implementation Notes
- Keep files under 50KB for fast loading
- Use .mp3 or .wav format
- Volume should be consistent across all sounds
- Consider adding volume controls in game settings
